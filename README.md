Data Structures and Algorithms implemented in Reason

# Contributing

First install `esy` (once)

```
npm install -g esy
```

Install (once, then when package.json changes)

```
esy install
```

Building / Testing

```
esy build && esy test
```

Or use watcher for development

```
esy watch
```
